# iOSReferenceRepository

### 概要
iOSのリファレンス実装をまとめたサンプルアプリです。

### プロジェクトのフォルダ構成
- 各言語別に階層を設ける
  - Swift
  - RxSwift
  - SwiftUI
- 画面ごとにディレクトリをつくる
  - MVCの例 
  - TopView
    - View
    - Repository
    - UseCase

### gitignoreするもの
- Pods/*
- *.xcworkspace

### 導入するライブラリ
- SwiftLint
- R.Swift
- RxSwift
- Alamofire
- AlamofireImage
- SwiftyUserDefaults(要検討)
